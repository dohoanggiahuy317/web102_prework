# WEB102 Prework - *Sea Monster Crowdfunding Game App*

Submitted by: **Huy Do**

**Sea Monster Crowdfunding Game App** is a website for the company Sea Monster Crowdfunding that displays information about the games they have funded.

Time spent: **2** hours spent in total

## 1. Required Features

### 1.1. The following **required** functionality is completed:

* [x] The introduction section explains the background of the company and how many games remain unfunded.
* [x] The Stats section includes information about the total contributions and dollars raised as well as the top two most funded games.
* [x] The Our Games section initially displays all games funded by Sea Monster Crowdfunding
* [x] The Our Games section has three buttons that allow the user to display only unfunded games, only funded games, or all games.

### 1.2. The following **optional** features are implemented:

* [x] List anything else that you can get done to improve the app functionality!

#### 1.2.1. Features Implemented

1. **Responsive Navigation Bar**
   - Fixed header with logo and navigation links
   - Smooth scrolling to sections when nav links are clicked

2. **Game Search Functionality**
   - Search input field and button added
   - Real-time filtering of games based on search query

3. **Visual Enhancements**
   - Improved color scheme and layout
   - Hover effects on game cards and stats containers
   - Animated transitions for interactive elements

4. **Content Organization**
   - Clear sections for About, Stats, and Games
   - Buttons to filter games (All, Funded, Unfunded)

5. **Accessibility Improvements**
   - Increased color contrast for better readability
   - Proper heading hierarchy

#### 1.2.2. CSS Changes

1. **Layout and Spacing**
   - Fixed header with `position: fixed`
   - Added `padding-top` to body to prevent content from being hidden under the header
   - Implemented flexbox for responsive layouts

2. **Color Scheme**
   - Updated to a more modern color palette (primary color: #3498db)
   - Improved contrast for text and backgrounds

3. **Typography**
   - Utilized 'Cabin' font family throughout
   - Adjusted font sizes and weights for better hierarchy

4. **Animations and Transitions**
   - Added hover effects to buttons and cards
   - Implemented smooth transitions for interactive elements
   - Funny animation is on Navigator bar

5. **Responsiveness**
   - Used relative units (%, em) for flexible sizing
   - Implemented media queries for different screen sizes (to be expanded)

#### 1.2.3. JavaScript Enhancements

1. **Search Functionality**
   - Added event listeners for search input and button
   - Implemented filtering logic for games based on search query

2. **Smooth Scrolling**
   - Added offset calculation to account for fixed header when scrolling to sections

3. **Dynamic Content Loading**
   - Improved functions for loading and filtering game data

#### 1.2.4. HTML Structure Updates

1. Added navigation bar to the header
2. Implemented semantic HTML5 elements for better structure
3. Added id and class attributes for JavaScript and CSS targeting
4. Included a search input field and button
